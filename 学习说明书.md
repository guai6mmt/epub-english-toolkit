# EPUB 英语学习系统说明书

这份说明书的目标是帮你把手头的大量 `epub` 英文期刊，稳定地转化为一套长期有效的英语学习流程，重点服务于：

- 提高英语阅读能力
- 提高英语口语表达能力
- 提高英语写作能力
- 积累国际新闻、商业、科技、文化背景知识
- 为雅思 8 分做长期准备

这套系统的核心思想不是“读很多”，而是：

- 用工具自动整理材料
- 用固定流程把一篇文章反复利用
- 把输入变成输出
- 用复习机制确保真正记住

## 2. 整个学习系统是怎么工作的

这套系统的学习链路是：

1. 导入一本 `epub`
2. 生成一份学习包
3. 每天看 `daily-plan`
4. 完成当天学习任务
5. 标记完成状态
6. 按系统安排做复习
7. 每周导出并整理成果

你可以把它理解成：

- `epub` 是原材料
- `study pack` 是一周课程表
- `daily plan` 是每天待办清单
- `tracker` 是完成记录
- `exports` 是可读、可复习、可迁移的学习资料

## 4. 每天应该怎么学

这里是最重要的一部分。系统只是辅助，真正提升来自你每天如何使用它。

建议你把每篇文章分成 4 个阶段：

- 阅读理解
- 词块整理
- 口语输出
- 写作输出

### 每天的推荐时长

如果你是长期准备雅思 8 分，建议：

- 工作日：`45-75 分钟`
- 周末：`90-120 分钟`

最低有效版本：

- 每天至少完成 `1 个 study task`
- 每天至少完成 `1 个 review task`

---

### 阶段 A：阅读理解

先根据 `daily-plan` 里的 `reading_focus` 来做。

对于一篇精读文章，推荐流程：

1. 先限时快速看一遍
2. 不查词，先判断主旨
3. 第二遍找结构
4. 第三遍再看细节和难句

你应该重点回答这几个问题：

- 这篇文章主要在讲什么
- 作者的核心观点是什么
- 作者用了哪些事实或例子支持观点
- 作者在哪一段从“描述事实”转向“表达判断”
- 如果这是雅思写作素材，它可以支持什么观点

不要一上来逐词查词。

正确做法是：

- 先理解段落功能
- 再理解句间逻辑
- 最后才处理真正影响理解的词

---

### 阶段 B：词块整理

系统提取的是 `phrase`，不是单纯单词，这一点非常重要。

你学习时要重点记住：

- 词块
- 搭配
- 可复用句型

例如，不要只记：

- `currency`

要记：

- `foreign currency`
- `short-term stability`
- `international pressure`
- `public opinion`

你需要做的不是“背很多”，而是：

- 每篇文章选 `5-8` 个最值得复用的词块
- 每个词块自己造 1 句
- 尽量在口语和写作里重新用出来

如果你想导入 Anki：

```powershell
python study_tool.py export-anki --pack-id te-2026-03-21-2026-03-31
```

导出文件：

- [Anki TSV](D:/BaiduSyncdisk/英语/TE/英语学习代码/exports/anki/te-2026-03-21-2026-03-31.tsv)

---

### 阶段 C：口语输出

这是从“会看”走向“会说”的关键。

每篇文章至少完成 3 件事：

1. `30 秒 summary`
2. `2 分钟 retell`
3. 回答 `Part 3` 风格问题

系统已经帮你生成了：

- `warmup`
- `retell`
- `part3_questions`
- `part2_cue_card`
- `shadowing_sentence`

推荐练法：

1. 先看文章，做简要笔记
2. 根据 `warmup` 说 30 秒
3. 根据 `retell` 说 2 分钟
4. 回答 1-2 个 `part3_questions`
5. 最后跟读 `shadowing_sentence`

如果你愿意加听力/语音训练，可以导出 TTS 文本：

```powershell
python study_tool.py export-tts --pack-id te-2026-03-21-2026-03-31
```

导出目录：

- [TTS 导出目录](D:/BaiduSyncdisk/英语/TE/英语学习代码/exports/tts/te-2026-03-21-2026-03-31)

如果你的 Windows 本地语音可用，也可以尝试：

```powershell
python study_tool.py export-tts --pack-id te-2026-03-21-2026-03-31 --audio
```

口语时最重要的原则：

- 不追求第一次就说得很完美
- 先追求说得出来
- 再追求结构清楚
- 最后再追求高级表达

---

### 阶段 D：写作输出

每篇文章至少做一种写作任务：

- `120 词摘要`
- `Task 2 风格议论文`
- `观点提纲`

系统里常见的写作任务有：

- `summary_120w`
- `essay_prompt`
- `task2_prompt`
- `outline_prompt`

推荐练法：

#### 轻量版

- 写 120 词摘要

适合平日时间少的时候。

#### 标准版

- 写摘要
- 再写一个 `Task 2 outline`

#### 冲刺版

- 写完整 250-320 词议论文

写作时请重点模仿：

- 论证结构
- 段落推进
- 对比表达
- 让步表达
- 客观正式语气

不要做的事：

- 只摘抄原句
- 只翻译中文思路
- 只堆积高级词而没有逻辑

## 5. 每周应该怎么学

系统默认是一份 5 天学习包，但你真正执行时，建议用 7 天节奏。

### 推荐的 7 天节奏

#### 周一

- 开始第 1 篇精读
- 做阅读理解
- 做 30 秒 summary

#### 周二

- 完成第 1 篇口语复述
- 写 120 词摘要
- 做当天复习

#### 周三

- 开始第 2 篇精读
- 提取 5-8 个词块
- 回答 1 个 Part 3 问题

#### 周四

- 完成第 2 篇口语 + 写作提纲
- 做第 1 篇的间隔复习

#### 周五

- 完成第 1 篇短文泛读
- 完成第 2 篇短文泛读

#### 周六

- 完成第 3 篇短文泛读
- 选一篇做 250 词写作

#### 周日

- 复盘整周
- 回看完成情况
- 整理本周最重要的：
  - 10 个词块
  - 3 个口语观点
  - 2 个写作论证框架

## 6. 如何标记完成

当你完成一个任务后，要立即标记，这样系统的复盘才有意义。

例如，完成第一个学习任务：

```powershell
python study_tool.py set-status --id "te-2026-03-21-2026-03-31:study:1" --kind study --pack-id te-2026-03-21-2026-03-31
```

```powershell
python study_tool.py set-status --id "te-2026-03-21-2026-03-31:review:the-americas-cuba-s-broken-economy-leaves-it-at-donald-trump-s-mercy:same_day_recall" --kind review --pack-id te-2026-03-21-2026-03-31
```

## 7. 如何看进度

你应该每周至少看 2 次进度报告：

```powershell
python study_tool.py progress-report
```

你需要重点看这几个指标：

- `study_tasks`
- `review_tasks`
- `completed_study_tasks`
- `completed_review_tasks`
- `study_completion_rate`
- `review_completion_rate`

建议目标：

- `study_completion_rate` 保持在 `0.8` 以上
- `review_completion_rate` 保持在 `0.7` 以上

如果阅读做了很多，但复习率很低，说明：

- 你输入很多
- 但沉淀不够

如果复习很多，但新的学习任务完成少，说明：

- 你节奏可能太重
- 需要调低每周任务量

## 8. 如何把这套系统用于雅思 8 分

你不是单纯在学“新闻英语”，而是在建立雅思 8 分需要的四种能力。

### 阅读

你要练的不是“全部看懂”，而是：

- 快速抓主旨
- 理解作者立场
- 识别逻辑结构
- 适应复杂长句

### 口语

你要练的不是“背答案”，而是：

- 用国际议题进行观点表达
- 用例子支撑观点
- 练正式但自然的表达

### 写作

你要练的不是“高级词堆砌”，而是：

- 明确立场
- 合理展开
- 使用有逻辑的句间衔接
- 通过文章积累真实素材

### 听力

虽然 `epub` 本身不是音频材料，但你可以通过：

- 跟读
- TTS 听读
- 复述录音

把“阅读材料”变成“听说材料”。

## 9. 最推荐的学习策略

### 策略一：一篇文章四次利用

一篇文章至少做四次：

1. 第一次：读
2. 第二次：说
3. 第三次：写
4. 第四次：复习

这是效率最高的方式。

### 策略二：少而精

不要一周读几十篇。

更好的方式是：

- 2 篇精读
- 3 篇短读
- 每篇都输出

### 策略三：词块优先

记住可复用表达，不要只背孤立单词。

### 策略四：输出优先于收藏

不要满足于“我看过了”。

真正的学习完成标准应该是：

- 我能复述
- 我能写摘要
- 我能回答观点题

### 策略五：长期积累主题知识

你的 `epub` 很适合积累这些高频主题：

- 政治与国际关系
- 商业与经济
- 科技与 AI
- 能源与环境
- 文化与社会变化

这些主题会反复出现在：

- 雅思口语 Part 3
- 雅思写作 Task 2
- 高级阅读与讨论

## 10. 你不应该怎么学

不要这样做：

- 每篇文章都查所有生词
- 只输入，不输出
- 只看中文对照
- 只收藏词汇，不复用
- 只做当天任务，不做复习
- 因为一篇太难就整周停下来

遇到太难的文章时，正确做法是：

- 保留主旨理解
- 只选部分段落做精读
- 把重点放在结构和表达，而不是完全读透

## 13. 最后的建议

如果你的目标是雅思 8 分，不要把这套系统当成“整理资料工具”，而要把它当成“英语输出训练器”。

真正让你进步的不是：

- 导入了多少本 `epub`
- 收藏了多少词
- 生成了多少任务

真正让你进步的是：

- 你是否稳定完成每天任务
- 你是否真的说出来了
- 你是否真的写出来了
- 你是否在一周后还能复述出来

所以你的目标应该始终是：

- 每周完成一份学习包
- 每天完成至少一个任务
- 每周至少有两次真实口语输出
- 每周至少有一次真实写作输出

只要你能连续执行 12 周，这套系统对你的英语能力和雅思备考都会有非常明显的帮助。
